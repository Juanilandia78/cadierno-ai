# Playbook - Pull Request

## Objetivo

Facilitar la revisión del código antes de integrarlo.

---

## Debe incluir

- Objetivo del cambio
- Cambios realizados
- Riesgos
- Evidencia de pruebas
- Capturas si aplica

---

## Checklist

✓ Compila

✓ Tests

✓ Documentación

✓ Sin conflictos

---

## Filosofía

Una Pull Request debe ser fácil de revisar.
