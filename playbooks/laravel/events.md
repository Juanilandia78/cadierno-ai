# Playbook - Events

## Objetivo

Desacoplar procesos mediante Events.

---

## Utilizar cuando

- Envío de emails
- Auditorías
- Notificaciones
- Integraciones

---

## Reglas

- Un Event representa algo que ocurrió.
- Los Listeners ejecutan acciones.
- Mantener bajo acoplamiento.

---

## Checklist

✓ Event

✓ Listener

✓ Registro

✓ Tests

---

## Filosofía

Los Events comunican.

Los Listeners actúan.
