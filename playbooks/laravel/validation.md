# Playbook - Validation

## Objetivo

Centralizar todas las validaciones utilizando Form Requests.

Nunca realizar validaciones complejas dentro de Controllers.

---

## Reglas

- Utilizar Form Requests.
- Centralizar mensajes personalizados.
- Utilizar Rules cuando agreguen valor.
- Mantener las reglas organizadas.
- Evitar duplicación.

---

## Checklist

✓ Existe Form Request

✓ Validaciones reutilizables

✓ Mensajes claros

✓ Autorización implementada

✓ Tests cuando corresponda

---

## Filosofía Cadierno AI

Validar temprano.

Validar siempre.

Nunca confiar en la entrada del usuario.
