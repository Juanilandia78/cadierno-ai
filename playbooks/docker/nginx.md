# Playbook - Nginx

## Objetivo

Configurar Nginx como servidor web y Reverse Proxy de forma segura y eficiente.

---

## Buenas prácticas

- No exponer archivos sensibles.
- Configurar correctamente PHP-FPM.
- Utilizar compresión cuando corresponda.
- Configurar Cache-Control para archivos estáticos.
- Registrar logs separados.

---

## Checklist

✓ Server Blocks correctos

✓ PHP-FPM conectado

✓ Logs habilitados

✓ Archivos sensibles protegidos

✓ Headers de seguridad

---

## Filosofía

Nginx debe ser rápido, seguro y transparente para la aplicación.
