# Playbook - PHP-FPM

## Objetivo

Ejecutar aplicaciones PHP de forma eficiente utilizando PHP-FPM.

---

## Buenas prácticas

- Utilizar la versión de PHP requerida por el proyecto.
- Instalar solo las extensiones necesarias.
- Configurar OPcache.
- Configurar límites de memoria adecuados.
- Mantener la imagen lo más liviana posible.

---

## Checklist

✓ Extensiones instaladas

✓ OPcache habilitado

✓ php.ini revisado

✓ Composer disponible

✓ Permisos correctos

---

## Filosofía

PHP-FPM debe ofrecer un entorno estable, reproducible y optimizado.
