# Playbook - Production

## Objetivo

Preparar una aplicación para ejecutarse en un entorno de producción de forma segura, estable y reproducible.

---

## Validar

- APP_ENV=production
- APP_DEBUG=false
- Variables sensibles protegidas
- Logs configurados
- Cache habilitada
- Queue Workers activos
- Scheduler configurado
- Backups automáticos

---

## Checklist

✓ Configuración revisada

✓ Secrets protegidos

✓ Permisos correctos

✓ Healthcheck

✓ Monitoreo

---

## Filosofía

Producción debe ser predecible, estable y segura.
