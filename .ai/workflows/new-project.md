# Workflow - New Project

## Metadata

**Prioridad:** Muy Alta

**Especialista Líder:** Architect

**Participantes**

- Documentation
- Backend
- Database
- DevOps

---

## Objetivo

Preparar Cadierno AI para trabajar correctamente en un proyecto nuevo.

---

## Flujo

### Paso 1

Leer:

- AGENTS.md

Si no existe:

Solicitar crearlo.

---

### Paso 2

Analizar automáticamente:

- Framework
- Lenguaje
- Versiones
- Base de datos
- Docker
- Infraestructura

---

### Paso 3

Generar automáticamente:

knowledge/project.md

---

### Paso 4

Detectar:

- Arquitectura
- Patrones
- Convenciones

---

### Paso 5

Crear:

knowledge/patterns.md

---

### Paso 6

Crear:

knowledge/decisions.md

---

### Paso 7

Actualizar AGENTS.md si detecta información faltante.

---

## Resultado esperado

Cadierno AI comprende completamente el proyecto antes de comenzar a desarrollar.
