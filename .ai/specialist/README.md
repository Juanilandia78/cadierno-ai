# Cadierno AI Specialists

Los Specialists representan un equipo virtual de ingeniería.

Cada uno posee una responsabilidad específica.

Ningún Specialist intenta resolver todo.

Cada uno aporta su experiencia y colabora con los demás para construir software de calidad.

## Principios

- Especialización.
- Colaboración.
- Simplicidad.
- Calidad.
- Comunicación.
- Mejora continua.
