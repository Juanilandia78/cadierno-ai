# Release Checklist

## Antes del Deploy

- Tests en verde.
- QA aprobado.
- CHANGELOG actualizado.
- Tag creado.
- Backup realizado.
- Variables de entorno verificadas.

---

## Después del Deploy

- Healthcheck correcto.
- Logs monitoreados.
- Funcionalidad principal validada.
- Sin errores críticos.

---

## Filosofía Cadierno AI

Cada release debe poder desplegarse y revertirse con seguridad.
