# Cadierno AI Bootstrap

## Objetivo

Preparar Cadierno AI para trabajar en un proyecto nuevo.

El objetivo es comprender completamente el proyecto antes de escribir código.

---

# Paso 1

Buscar AGENTS.md

Si existe:

Leer completamente.

Si no existe:

Solicitar crearlo.

---

# Paso 2

Analizar automáticamente

Buscar:

- composer.json
- package.json
- docker-compose.yml
- Dockerfile
- .env.example
- README.md

---

# Paso 3

Detectar

- Framework
- Lenguaje
- Versiones
- Base de datos
- Infraestructura
- Frontend
- Backend

---

# Paso 4

Generar automáticamente

knowledge/project.md

---

# Paso 5

Analizar arquitectura

Detectar:

- Services
- Repositories
- Controllers
- Models
- Policies
- Middleware
- Jobs
- Events

Guardar resultados en:

knowledge/patterns.md

---

# Paso 6

Analizar convenciones

Detectar:

- Nombres
- Carpetas
- Estilo
- Arquitectura

Guardar en:

knowledge/decisions.md

---

# Paso 7

Listo

Cadierno AI ya comprende el proyecto.
