# Cadierno AI Bootstrap

## Objetivo

Inicializar Cadierno AI dentro de cualquier proyecto.

Antes de escribir una sola línea de código, el proyecto debe ser completamente comprendido.

Nunca asumir tecnologías, arquitecturas o convenciones.

Primero analizar.
Después comprender.
Finalmente trabajar.

---

# Fase 0 - Verificación

Verificar si Cadierno AI ya fue instalado.

Buscar:

- .ai/
- knowledge/
- AGENTS.md

Si existe una instalación previa:

- actualizar el conocimiento;
- conservar documentación existente;
- no sobrescribir archivos sin autorización.

Si no existe:

continuar con la instalación.

---

# Fase 7 - Generación del conocimiento

Copiar automáticamente los templates ubicados en:

templates/knowledge/

Crear:

Generar y completar automáticamente:

- project.md
- architecture.md
- decisions.md
- patterns.md
- integrations.md
- technical-debt.md
- developer.md

Toda la información debe obtenerse analizando el proyecto.

Nunca inventar información.

# Fase 8 - Activación de Specialists

A partir de este momento todos los Specialists deberán utilizar la información ubicada en:

knowledge/

Antes de responder.

Si la información no existe, ejecutar nuevamente Bootstrap.

# Fase 9 - Proyecto listo
